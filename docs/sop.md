# 产品
## 介绍
该产品是基于WXT框架开发的浏览器插件，主要作用是辅助渗透测试工程师进行网络安全测试
## 特点
### 抓包/重放
该插件会在F12面板中注册界面,用户可以通过F12选择抓包模块,只要刷新网页就可以获取到捕获的数据包,并展示在request和response面板中
### payload存储
在F12中，有payload存储模块，该模块保存了渗透测试常用的payload，类似hackbar。用户可以在该模块选择复制payload
### 数据生成
在F12中，用户可以选择数据生成模块，生成渗透测试常用的数据
### JS 端点获取
在popup面板中，用户可以看到插件捕获的路径
